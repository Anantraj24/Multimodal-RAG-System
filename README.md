# 🚀 Multimodal RAG System

A modern, full-stack real-time Multi-Modal Retrieval Augmented Generation (RAG) system with a beautiful, responsive user interface designed in React and Vite.

**👉 Live Demo:** [https://ntroragsystem.netlify.app/](https://ntroragsystem.netlify.app/)

## ✨ Key Features

- **Multi-Modal Understanding:** Process and retrieve information intelligently combining different types of data.
- **Modern UI:** Designed with a stunning interface using Radix UI primitives and Framer Motion animations.
- **Lightning Fast:** Built on top of Vite and React, ensuring rapid development and excellent production performance.
- **Fully Responsive:** Functions seamlessly across mobile, tablet, and desktop environments.

## 💻 Tech Stack

- **Frontend:** React 18
- **Build Tool:** Vite 6
- **Styling:** Vanilla CSS / Tailwind CSS
- **Components:** Radix UI Primitives, Lucide Icons
- **Animations:** Framer Motion

## 🚀 Getting Started Locally

### Prerequisites

Ensure you have [Node.js](https://nodejs.org/) installed on your machine.

### Installation

1. Clone the repository
2. Install the necessary dependencies:
   ```bash
   npm install
   ```

### Running the Development Server

Start the local Vite development server:
```bash
npm run dev
```

The application will be available at `http://localhost:3000`.

## 📦 Building for Production

To create an optimized production build:
```bash
npm run build
```
This will compile the application and output it into the `build` directory, ready to be hosted on platforms like Netlify or Vercel.
